# Intel<sup><font size=2>®</font></sup> HPC Toolkit
## Deprecation Notice
With the 2026.0 Release, the Intel® oneAPI Base Toolkit (Base Kit) and Intel® oneAPI HPC Toolkit have been combined into the Intel® oneAPI Toolkit. <br />
You can still use the Dockerfiles, but they will not be updated starting with version 2026.0.

### **Optimized Tools for High-Performance Computing**

The Intel® HPC Toolkit delivers what developers need to build, analyze, optimize, and scale high-performance computing (HPC) applications with the latest techniques in vectorization, multithreading, multi-node parallelization, and memory optimization. [HPC Kit Details](https://software.intel.com/oneapi/hpc-kit)
