# Intel<sup><font size=2>®</font></sup> Fortran Essentials
## Deprecation Notice
With the 2026.0 Release, the Intel® Fortran Essentials and Intel® C++ Essentials have been combined into the Intel® oneAPI Toolkit. <br />
You can still use the Dockerfiles, but they will not be updated with 2026.0 and beyond.

### **Essential Kit for Fortran developers**

The Intel® Fortran Essentials image includes tools for compiling and debugging Fortran HPC code.
